    R4D1

    Within this project we delved into the Rijksmuseum to check whether the overall design could be improved.

    Using mainly quantitative and heuristic type of reserach and analysis we uncovered several issues with the design.

    Overall, few things could be improved but are related to user actions, navigation and efficiency/ simplification of controls.

    Heuristics analysis showed that the museum's digital studio, closely resembled the mechanics of Pinterest.

    The search engines of the studio proved to be difficult and frustrating to use for specific tasks.

    The most flawed element within the design proved to be problems with the search engine & navigation between user settings.
